# raw/ — the inbox

Anything that's not yet filed lands here. Brain dumps, meeting transcripts, screenshots, downloaded reports, half-baked ideas.

## Rules

- File naming: `YYYY-MM-DD_topic-slug.md`
- No subfolders. Flat list. Date-prefix sorts chronologically.
- Compile out to `wiki/` or `output/` weekly.
